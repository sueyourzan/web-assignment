📖 课程简介

课程概述

2025年9月 · 软件工程专业

本课程是软件工程专业的一门专业必修课。通过对Web前端三大主流开发技术学习、探究和实践，让学生理解和掌握HTML5、CSS3、JavaScript等脚本语言的相关知识，通过项目实战和训练培养学生开发和设计Web网站或网页的基本操作技能和工作岗位适应能力。

通过本课程的学习，使学生理解和掌握HTML（超文本标记语言）中各种常用标记的使用；CSS（层叠样式表）常用属性的使用，如何运用 div + CSS 对网页进行布局；JavaScript脚本语言的基础知识，以及借助于JavaScript语言的强大功能增强网页功能，实现用户交互事件等动态网页功能，为后续课程奠定学习基础。

📋 学习目标

2025-2026 第二学期

通过本课程的学习，学生应达到以下目标：

1. 熟练掌握 HTML5 语义化标签的使用，能够构建结构清晰的网页文档；

2. 掌握 CSS3 布局技术（Flex、Grid、浮动定位等），能够实现美观的页面样式；

3. 理解 JavaScript 核心概念，能够实现页面交互与动态效果；

4. 具备响应式设计能力，能够开发适配多终端设备的Web页面。
